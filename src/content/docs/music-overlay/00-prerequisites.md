You will need to make sure you have/can access a few things before using the overlay!

## Prerequisites

- The **music source** you will be playing/streaming music from.
  - As of writing this can be one of: YouTube Music (via [Pear Desktop](https://github.com/pear-devs/pear-desktop/)), Spotify, Apple Music (via [Cider](https://cider.sh)), or Last.fm.
  - Not all sources are made equal! Some require a paid subscription and app purchase (apple music/cider), some need a sub for API access (*ahem* ***SPOTIFY***... you mfers)
- Your streaming software/widget destination of choice (probably OBS)
- A functional browser
- Ability to read
- Removed herobrine
