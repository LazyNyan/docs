## On SoundCloud integration

frick u soundcloud

~~It is not possible for me to directly integrate soundcloud for the following reasons:~~

*Technically* it is possible via the *undocumented API used by the soundcloud.com website*, but all of the points below still apply.

- You need an Artist Pro subscription to even [create an API integration](https://soundcloud.com/you/apps/new). lame approach imo.
- Their [Public API terms](https://developers.soundcloud.com/docs/api/terms-of-use) seem stricter than any of our currently supported sources
- The currently existing [Public API routes](https://developers.soundcloud.com/docs/api/explorer/open-api#/me/get_me_feed_tracks) would make it less reliable for a now playing widget to obtain *only* song data, e.g. if one also does actions like reposting, etc.

But you can continue scrobbling SoundCloud to Last.FM with no additional limitations.

**Update:** Soundcloud is now supported via the aforementioned methods, but it may unexpectedly break in the future due to the nature of undocumented website apis :p
