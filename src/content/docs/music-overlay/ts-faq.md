## Can you support [insert site]?

If it has an accessible API, maybe.

If it doesn't have an API, your best bet is scrobbling songs from there to LastFM if you can.

## On SoundCloud integration

frick u soundcloud

It is not possible for me to directly integrate soundcloud for the following reasons:

- You need an Artist Pro subscription to even [create an API integration](https://soundcloud.com/you/apps/new)
- Their [regulation](https://developers.soundcloud.com/docs/api/terms-of-use) seems far stricter than any of our currently supported sources
- The currently existing [API routes](https://developers.soundcloud.com/docs/api/explorer/open-api#/me/get_me_feed_tracks) would make it less reliable for a now playing widget to obtain *only* song data, e.g. if one also does actions like reposting, etc.

But you can continue scrobbling SoundCloud to Last.FM with no additional limitations.
